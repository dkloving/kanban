# Phase 09 Plan: Keyboard Speed UX

## Goal
Enable fast keyboard-first operation for core board flows.

## Scope
1. Shortcut for new task.
2. Arrow-key navigation between cards.
3. Search palette to focus and open a task.
4. Enter to open and Escape to close overlays.
5. Shortcut hints in UI.

## Out of Scope
1. New orchestration features unrelated to keyboard control.

## Test Gate
1. Create and start a task using keyboard only.
2. Navigate across cards with arrow keys.
3. Open search palette, select task, open detail, close with Escape.

## Exit Criteria
1. Core workflow is fully usable without mouse.
2. Shortcut behavior is consistent and discoverable.
